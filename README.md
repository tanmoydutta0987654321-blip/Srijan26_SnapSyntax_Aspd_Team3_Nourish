# Nourish-Home-Cooked-Food
hackathon project for JU
🍱 Smart Tiffin Booking System

A modern, responsive single-page web application for ordering daily meals and managing subscriptions.
Designed for students and working professionals who need quick, affordable, and reliable meal options.

---

🚀 Live Demo

(https://tanmoydutta0987654321-blip.github.io/Srijan26_SnapSyntax_Aspd_Team3_Nourish/)

---

📌 Features

- 🏠 Home Section
  
  - Clean landing page with banners and categories
  - Highlights popular meal providers

- 🍛 Menu Section
  
  - Browse meals (Veg, Non-Veg, Healthy, Budget)
  - View pricing and add items to cart

- 🛒 Cart System
  
  - Add/remove items
  - Update quantity
  - Auto total calculation
  - Persistent data using localStorage

- 📦 Subscription Plans
  
  - Weekly & monthly meal plans
  - Easy selection UI

- 🚚 Order Tracking
  
  - Step-based delivery status (Order → Cooking → Out → Delivered)

- 🔄 Single Page Navigation
  
  - Smooth transitions between sections
  - No page reloads (SPA-like behavior)
